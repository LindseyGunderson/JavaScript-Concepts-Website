What is jQuery?

JQuery is a JavaScript library which is used to make the use of JavaScript easier on a website. Jquery can take lines of JavaScript code and simplify complex code blocks so Jquery has many built in functions for DOM manipulation, selection, events and even incorporate animations. A couple of advantages to using JQuery are similar to JavaScript so it can be easy to learn, increase in development time because you’re writing less code plus its more readable and you can also include the library remotely for a lightweight option. Google’s API can easily be linked within a script tag at the beginning of your html file.

In the next few sections we will focus on: Syntax, Document Object Model and Animations.

1.1 - jQuery selector syntax and selectors

The syntax for JQuery is similar to JavaScript but is condensed for readability. Let's quickly compare the differences with syntax:

JavaScript JQuery
document.method(selector) $(selector).action()

As you can see, they are pretty similar in synax.

  • The $ defines jQuery and starts our statement.
  • The selector will be the element we are locating or trying to find.
  • The action() refers to the action that we want to preform on the element.

    • Let's see how this works in an example:

    JavaScript JQuery
    document.getElementById("firstname").value; $("#firstname").val();

    In the example above, we are selecting the element by the id that has the name "firstname". They will both return the same value but we should note the # in the jQuery statement. The "#" is used when targetting a id and a "." is used when you are targetting a class. Either the "#" or "." must be within the selector or jQuery will not know what element to grab.
