What is JavaScript?

JavaScript is a programming language that allows you to bring more dynamic interactions to the html of your website. This can be used both on client side and server side for more a more engaging website to a user. HTML and CSS can help bring the structure to your website but with the additional implementation of JavaScript, you can open even more opportunities for interactivity.

1.1 - What is a function in JavaScript?

Functions are a great option to use in your code for reusability if you are planning on repeating a task multiple times in any coding language. They are executed when they are called by the defined name given to them which is also known as invoking. You can define your own function by the key word “function”, followed by the name of your funnction, then brackets (which can have parameters - more in the next section) and the code statments enclosed in curly braces.

  • The following is the syntax for declaring a function: 
function functionName(){  statements  }

1.2 - What are parameters in JavaScript?

To take this a step further, we can also pass parameters into functions to use within the function itself.

  • The synax is similar to the previous section:

Single parameter

function functionName(parameter){  statements  }

Multiple parameters

function functionName(parameter, parameter){  statements  }

You can also pass multiple parameters and separate each witha comma. When declaring parameters for functions, their names should be unique and are descriptive. Its also good to keep in mind parameters passed and are within the function, will be local scope. This means, once our function is done running, the parameters will be forgotten until the function is called again.

2.1 - What are conditional structures?

We use conditional structure statements to test if a particular condition is true or false. There are a few different options for conditionals depending on the action. The most common conditional statements are:

  • if
  • if else
  • else if
  • switch case

2.2 - What are if statements?

If statements are used to check if a condition is true. The written syntax for an if statement is as follows:

2.5 - What are switch statements?

That’s a lot of code in the previous section… there has to be a better way. One option we can also dive into is a switch statement. Switch statements work nicely when you’re checking against many possible options.

Advantages:

  1. Easy to read to the human eye and if any conditions might have been missed.
  2. Simple to add more options if needed for case checks.
  3. You can add multiple groupings to one single case.

Limitations:

  1. The expression will only be evaluated once for the entire statement.
  2. There needs to be an exact match when the cases are checked.

How do we define a switch statement:

            switch (expression) {
                case NUMBER1:
                      // statement if matches NUMBER1
                      break;
                case NUMBER2:
                case NUMBER3:
                    // statement for NUMBER2 & NUMBER3
                    //this is an example of grouping statements
                      break;
                case NUMBER4:
                    // statement for N4
                      break;
                default:
                    // default statement for anything 
                    //that doesn't match previous.
                      break;
                  }

What is an array or collection?

An array or collection is a type of variable that can hold many types of values within one variable. Keep in mind, arrays should hold variables of the same type within their collection. This could be a collection of strings, integers, or other variable types used in JavaScript.

How do we create arrays?

We can declare an array by using the following: 

var newArray = [];

Where "newArray" will be the name we define for our array. We add items to an array manually but placing the items within "" and in the []. We can use strings, numbers and even objects within arrays for more complex code!

Let's start simple and make an array of fruits. 

var fruits = [‘apples’, ‘oranges’, ‘banana’, ‘watermelon’];

Awesome! We now have our array of fruits ready to go. Its also important to note that when you want to display a particular item from within the array, you can access it by its index value. The index refers to the position of the value in the array. To be a little tricky, array index values start at 0 and that can often be forgotten. Let's say we want to access the oranges item from the array. 

console.log(fruits[1]);

We would see the item "oranges" in our console and we can do the same for the other fruits. Arrays also have a few Document Object Model (DOM) methods built in that we can use to manipulate or find out more information about arrays or collections. Some of those methods include:

  • Array.length: it will give us the total length of the array (number of items within the array).
  • Array.push(): We can add something to the end of the array by “pushing” it.
  • Array.pop(): It will remove the last item in an array.
  • Array.shift(): this will remove the first item from the array list.
  • Array.sort(): will sort the items inside the array in ascending order.

How to traverse an array?

Traversing an array will print all the values within the array if we are not looking for a particular index value and just want to see what is in our array collection. We can do this with a simple for loop and it will do most of the work for us. 

for (initialization; condition; increment/decrement) { // code to be executed }

See the Pen traverse array by Lindsey ( @LGundy) on CodePen.


The for loop will iterate over each item within the array and we have to keep in mind that arrays start with a 0 index. Therefore, we start at zero and end with the array minus 1. Once the loop is at the end of the array, it will break out of the loop and print the results to the console.

**Note: Be careful when writing out loops, be sure that you double check your conditions and statements. Speaknig from experience, its very easy to end up in an infinite loop which will potentially crash your program. Starting with i as our counter, this will be used as a counter for our array and determine when the end of the array. Once i is less than the array.length value (which is 4 in our case) It will stop looping through the array and the values will be printed to the console.
